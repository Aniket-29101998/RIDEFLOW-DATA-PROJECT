RideFlow Analytics is a comprehensive SQL project analyzing ride-booking data. 
It includes 10 key queries to extract business insights from a ride-sharing database, 
covering successful bookings, vehicle performance, customer/driver behavior, payment trends, and cancellation analysis.

## Key Queries:
- Retrieve successful bookings and total booking value
- Average ride distances and ratings by vehicle type (Prime Sedan, etc.)
- Top customers by ride count
- Cancellation stats by drivers (personal/car issues) and customers
- UPI payment rides and incomplete ride reasons
- Max/min driver ratings for premium services

Built with MySQL, perfect for showcasing SQL skills in data analysis and business intelligence. Dataset imported via MySQL Workbench (100k+ rows, 14 columns).
